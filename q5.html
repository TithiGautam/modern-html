Features of HTML5

Feature 1:Semantics

HTML5 introduces new semantic elements like header,nav,section etc., providing a clearer structure to web documents and enhancing accessibility.

Feature 2:Multimedia Support

HTML5 includes native supports for audio,video elements, reducing the reliance on third party plugins like flash.
This makes it easier to embed multimedia content directly into web pages.

Feature 3:Responsive Design Support

HTML5 encourages the development of responsive web design with features like meta, viewport tag and media queries,facilitating better adaptation to various screen and devices.
